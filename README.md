### Getting Started

##### 1. Clone repository.

```
$ git clone https://github.com/achalvs/synapse-frontend && cd synapse-frontend
```

##### 2. Install `npm` packages.

```
$ npm install
```

This may take a few minutes.

##### 3. Start the process in developer mode (hot-reloading).

```
$ npm run dev
```
